# Non-Home Files

Here we put files that we need to deploy outside the home directory. (Chezmoi
does not want to do this:
https://www.chezmoi.io/user-guide/frequently-asked-questions/design/#can-i-use-chezmoi-to-manage-files-outside-my-home-directory)

- `keyboard-backlightd`: Configuration for my Lenovo ThinkPad Z13's keyboard
  backlight daemon.
- `reflector.conf`: Configuration for the `reflector` package, which retrieves
  and filters the latest Pacman mirror list.
- `thinkfan.conf`: Configuration for the `thinkfan` package, which controls the
  fan speed on my Lenovo ThinkPad Z13.
